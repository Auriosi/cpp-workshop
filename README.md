# A Hacker's Guide to C++

Welcome to the HackKU26 Beginner Series C++ Workshop! This repository contains all the materials used during the workshop, which includes
* The source code for the Hackathon Survival Simulator (/final)
* A short guide to setting up your C++ development environment (tooling.md)
* The presentation slides used during the workshop (A Hacker's Guide to C++.pptx)
* A curated list of resources for further learning and development (resources.md)
